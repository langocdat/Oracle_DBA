# 1. Chuẩn bị trước khi patch
## 1.1 Tạo folder chứa file patch
[root]:
```
mkdir -p /u01/software
chown oracle:oinstall /u01/software
chmod 775 /u01/software
```

Attention: Đối với Patch OJVM, cần shutdown toàn bộ Instance.

## 1.2 Unzip file patch và OPatch
- Đảm bảo toàn bộ file OPatch trong GRID HOME và ORACLE HOME được thay thế bằng phiên bản mới
- Backup và giải nén OPATCH

[root]:
```
mv OPatch OPatch.bkp
```

# 1.3 Backup DB_HOME và oraInventory
[root]:
```
[root@misdb01 ~]# cd /u01/app/oracle/product/19c
[root@misdb01 19c]# tar -pcvf dbhome_1_backup.tar dbhome_1

[root@misdb01 ~]# cd /u01/app
[root@misdb01 app]# tar -pcvf oraInventory.tar oraInventory
```

## 1.4 Chạy OPatch Check Conflict
```
[oracle@misdb01 ~]$ . oracle
[oracle@misdb01 ~]$ $ORACLE_HOME/OPatch/opatch prereq CheckConflictAgainstOHWithDetail -phBaseDir /u01/software/38906621
```

# 2. Tiến hành Patch OJVM

## 2.1 Run apply Patch

```
[oracle@misdb01 ~]$ cd /u01/software/38906621
[oracle@misdb01 36199232]$ /u01/app/oracle/product/19c/dbhome_1/OPatch/opatch apply
[oracle@misdb01 36199232]$ /u01/app/oracle/product/19c/dbhome_1/OPatch/opatch lsinventory
```

- Startup all Instance

```
[oracle@srvdc OPatch]$ ./datapatch -verbose
```
